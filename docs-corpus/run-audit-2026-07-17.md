## Biggest Version Conflicts

- Express 4 vs Express 5: async Promise rejection forwarding and route path matching differ. Our repo is Express 4.18.2.
- MongoDB 5.0 vs 5.1+: `$lookup` with sharded foreign collections starts in 5.1, but our container is MongoDB 5.0.
- React 18 vs React 19: actions and `useActionState` are React 19-era docs, while our app is React 18.2.0.
- Node 18 vs Node 20.6+: built-in `--env-file` support is Node 20.6+, while our backend uses Node 18 and `dotenv`.
- MongoDB Node driver 5 vs older examples: callbacks were removed from the public API, while repo code still uses a callback-style `findOne()`.
